    Who We Are
    A Redwood Center for Theoretical Neuroscience research group is responsible for development of first version of NWB data format. A next step is a development of query engine that facilitates accessing data stored in the NWB format.
    Our Mission
    Neurodata Without Borders is going to be a standarized way of storing neurophysiology data. Once data is stored a suitable mechanism for its accessing must be provided. As a solution we implemented a query engine that enables executing of easy-to-understand queries on a set of files. This web page facilitates working with the query engine by providing a simple user interface to stored data. When a query is executed and desired values are obtained a user can even download a file with data stored.
    Query Examples
    Query Description
    analysis:(description LIKE whisker) selects all datasets description from an analysis group which contains a whisker string
    processing:(electrode_idx>30) selects all electrode_idx datasets from a processing group which value > 30
    epochs:(start_time>200 & stop_time<400 | stop_time>1600) selects all epochs which start_time > 200 and stop_time < 400 or stop_time > 1600
    epochs:(start_time) selects all epochs with a start_time dataset
    data:(unit LIKE %unkno%) selects all datasets data with attributes containing a substring unkno
    pole_in/data:(unit LIKE unkno) takes into account only data within a pole_in group
